            Q4. "Print output: 
            var x = 21;
            var girl = function () {
                console.log(x);
                var x = 20;
            };
            girl ();"
        
    
        
        The output of the code will be undefined.

        Explaination:
        var x = 21;: Here, a variable x is declared and initialized with the value 21
        console.log(x) It tries to print the value of the variable x. 
        However, there is a issue due to hoisting.
        var x = 20  Inside the function, a new variable x is declared and 
        initialized with the value 20. 
        This declaration takes priorty due to hoisting.

        "hoisting," means that variable declarations are moved 
        to the top of their containing scope during the compilation phase. 
        However, only the declarations are hoisted, not the initializations.

Q7. Write a function that accepts parameter n and returns factorial of n

        function factorial(n) {
            if (n === 0 || n === 1) {
              return 1;
            } else {
              return n * factorial(n - 1);
            }
          }
    
          const result = factorial(5);
          console.log(result); // Output: 120
